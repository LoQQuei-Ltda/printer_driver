===============================================
SISTEMA DE GERENCIAMENTO DE IMPRESSÃO
===============================================

Versão: 1.0
Desenvolvido por: LoQQuei-Ltda

VISÃO GERAL
-----------
O Sistema de Gerenciamento de Impressão é uma solução completa para administrar 
suas impressões em rede, permitindo gerenciar arquivos, monitorar status de impressoras
e controlar trabalhos de impressão a partir de uma interface simples e intuitiva.

REQUISITOS DO SISTEMA
---------------------
- Windows 10 versão 1903 ou superior
- Pelo menos 4GB de RAM
- Pelo menos 20GB de espaço livre em disco
- Virtualização habilitada na BIOS/UEFI
- Conexão à internet (para instalação)

INSTALAÇÃO
----------
1. Execute o aplicativo a partir do ícone na área de trabalho ou menu iniciar.
2. Faça login com suas credenciais (fornecidas pelo administrador).
3. Na tela principal, clique no botão "Instalar Sistema".
4. Siga as instruções na tela para concluir a instalação.
5. Após a instalação, reinicie o aplicativo se necessário.

NOTA: A instalação requer privilégios de administrador e pode levar alguns minutos.

UTILIZAÇÃO
----------
1. Inicie o aplicativo a partir do ícone na área de trabalho ou menu iniciar.
2. Faça login com suas credenciais.
3. Na tela principal, você verá a lista de arquivos disponíveis para impressão.
4. Clique no botão "Atualizar Lista" para atualizar a exibição.
5. Para imprimir um arquivo, clique no botão "Imprimir" ao lado do arquivo desejado.
6. Selecione a impressora e clique em "Imprimir".
